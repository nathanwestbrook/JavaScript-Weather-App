# JavaScript-Weather-App

This is an app that uses the **OpenWeatherMap** API

You can enter:
* Specific City Name (Might need country as well)
* Zip code

It will provide you with:
* City Name
* Exact Date
* Current Temperature
* Current Forecast
* Today's High/Low Temperature
